## [0.1.0] - 2024-12-04

### Added
- Initial release of Hotpaws
- F19 hotkey activation
- Fullscreen overlay with command categories
- Click-to-insert command functionality
- Support for Terminal.app, iTerm2, and Warp
- WKWebView-based interface
- HTML/CSS/JS overlay
- Swift + Cocoa architecture
- Universal binary (Intel + Apple Silicon)
- AppleScript terminal integration
- Customizable commands via ~/.hotpaws/commands.json
- Customizable styles via ~/.hotpaws/style.css
- ConfigManager for user config loading
- Build script for universal binary compilation
- Complete project documentation

### Known Issues
- Requires manual Accessibility permissions grant on first run
- No onboarding flow for permissions
